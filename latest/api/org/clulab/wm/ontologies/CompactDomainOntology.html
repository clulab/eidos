    leafStrings

    All the strings used in the leaves of the ontology

    leafStringIndexes

    Indexes into leafStrings sorted by leaf node

    leafStartIndexes

    Where to start in leafStringIndexes to find the indexes for leaf node N

    patternStrings

    All the regex strings used in the leaves of the ontology

    patternStartIndexes

    Where to start in patternStrings to find the patterns for leaf node N

    nodeStrings

    All the strings used in the non-leaf nodes of the ontology

    leafIndexes

    Parent offset, name offset, parent offset, name offset, ... for leaves only Name offset is into nodeStrings, parent offset is into branchIndexes.

    branchIndexes

    Parent offset, name offset, parent offset, name offset, ... for non-leaves only Name offset is into nodeStrings, parent offset is back into branchIndexes.
