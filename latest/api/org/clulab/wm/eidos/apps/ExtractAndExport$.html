ExtractAndExport 

object ExtractAndExport extends App with Configured

App used to extract mentions from files in a directory and produce the desired output format (i.e., jsonld, mitre tsv or serialized mentions). The input and output directories as well as the desired export formats are specified in eidos.conf (located in src/main/resources).
